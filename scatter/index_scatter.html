Scatter Plot

In this article we will see how the interaction with the data can transfer a five-dimensional data to the two-dimensional plane in an understandable way.

The content of the data is not of interest to us at the moment, but we can say that the data consists of two continuous and three discrete dimensions. The relation between continuous variables will be shown on the x and y axis and the discrete dimensions will be shown using radius, color and tooltips.

Now expand the figure by clicking and dragging the solid triangle shape you see on the right side of the figure. You can aslo drag the legend and titles of dimensions if you wish. You can see that you can classify the data according to the continents by hovering the mouse over the small colored rectangles in the legend section. And also you can see the country names by hovering the mouse over circles.

Now I want you to answer some questions

Which continent seems to have a better human index and income, also which country has the highest of both index?
Which country has the worst of both index?
In which continent the level of income does not increases as the development index increases?
Which continent has the highest population countries?
Does the income increase or decrease with human index as the population increase?
Which country is most suprising to you?

My answer to the last question is USA because it has a higher population than other countries with the similar index.

My point is you can answer a lot of questions by drawing and interacting with data. While you answer each question you see the whole figure at the same time which makes you understand the big picture and its parts much easier. Imagine you just write some query and fetch the answer of each question, this only helps you to answer your questions but it will not allow you to comprehend the whole data and make comparisons and inferences.

Data exploration should give you a chance to interact with data.

Lets see the main function of our script

let xaxis = new XAxis()
xaxis
  .header("Human Development Index(HDI)")
  .range(xrange)
  .show_domain(false)
  .margin(20)

let yaxis = new YAxis()
yaxis
  .header("Annual GDP per capita(USD)")
  .range(yrange)
  .show_domain(true)
  .margin(20)
const color_map = d3.scaleOrdinal(d3.schemeDark2);

let legend = new Legend()
  .labels(legend_labels)
  .colors(legend_colors)

let figure = new Figure()
figure
  .id("hdi")
  .width(400)
  .height(400)
  .xaxis(xaxis)
  .yaxis(yaxis)
  .grid(true)
  .legend(legend)
  .dimx(dataset.map(d => d.hdi)
)   .dimy(dataset.map(d => d.gdp))
  .dimr(dataset.map(d => d.population))
  .dimc(dataset.map(d => d.continent).map(d => color_map(d)))
  .dimtt(dataset.map(d => d.name))
  .translate(50, 50)
  .draw()

As seen in the code, we start by creating axes first, then we create the legend and at last we create our figure. We provide five different data that will be included in the figure.

Data we provide to the dimx and dimy function represents the data shown on the x and y axis, dimr is for the radius of circles, dimc is for the color and dimtt is for tooltip.

The rest is drawing lines, circles and rectangles. If you want to see how this figure was drawn, you can look at the source code of this page as always.
